# Requirements

## Linux packages:

#### Main app:
 - python3 with:
    - `python-can`
 - python-dbus
 - BlueAlsa (Recommend to build with AAC (iPhone/Andorid) or aptx (Android) support)
 
#### Screen Reader (screen-reader.py)
 - python3 with:
    - `bitstring`
 
## Hardware:
 - Can Module (`can0` interface)
 - Audio card
 - Bluetooth module (Can be built in)
